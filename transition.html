2b
"CSS Transitions allow property changes in CSS values to occur smoothly over a specified duration."
ease: Transitions with slow start, then fast, then slowly ends.
linear: Transitions at a linear/steady speed from start to end
ease-in: Transitions slowly when it starts.
ease-out: Transitions out slowly.
ease-in-out: Transitions slowly in and slowly out.
cubic bezier: You can set 4 values (n,n,n,n) in a cubic-bezier function.
